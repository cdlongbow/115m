# 115m 当前功能清单

## 核心能力

- 列表页视频预览图：在 115 文件列表中为视频项生成并展示多帧预览图。
- 点击视频播放：点击视频文件名直接由扩展接管播放流程。
- 无损优先播放：默认优先使用无损（Ultra）地址开播。
- 原画降级兜底：无损不可用时自动回退到 115 原画 m3u8 播放链路。

## 播放器能力

- 画质切换：支持无损与 115 原画切换。
- 播放记忆：自动保存与恢复播放进度。
- 基础快捷键：空格播放/暂停，方向键快进快退和音量控制，F 全屏。
- 进度预览：进度条悬停显示时间与缩略图预览。
- 错误兜底：主链路失败时自动尝试降级并给出错误提示。

## 性能与稳定性

- 预热机制：点击前后触发视频源预热，减少首播等待。
- 缓存机制：background 内存 + session 缓存预热结果。
- 并发去重：同一 pickCode 请求复用 inflight，减少重复请求。
- 开页防抖：短时间重复点击同一视频时去重，避免多标签页并发打开。

## 下载与鉴权

- 下载地址获取：优先 Pro 接口，失败降级 Web 接口。
- Cookie 同步：拦截 token 并写入下载域 cookie，保障播放鉴权链路。

## 当前工程约定

- 正式构建命令：`pnpm build`
- 打包命令：`pnpm zip`（仅用于生成发布压缩包）
- 发布目录：`release/`（已在 `.gitignore` 中整体排除）
