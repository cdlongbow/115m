# 115m

这是一个 115 网盘增强扩展。你只要记住下面这些就够了。

## 现在有什么功能

- 列表页视频有预览图
- 点击视频直接进入扩展播放器
- 默认优先无损播放，失败会自动回退到 115 原画
- 支持画质切换（无损 / 115原画）
- 支持播放进度记忆
- `hls.js` 按需加载，不影响默认无损首播

## 常用命令

```bash
pnpm install
pnpm build
pnpm zip
```

- `pnpm build`：开发后验证是否可用
- `pnpm zip`：打包发布用

## 本地使用

1. 打开 `chrome://extensions/`
2. 开启开发者模式
3. 加载 `dist/` 目录
4. 改代码后执行 `pnpm build`，然后在扩展页点重新加载
