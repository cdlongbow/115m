# AGENTS.md

这是 `115m` 项目的工作规则。

## 项目

- 项目名：`115m`
- 这是一个 Chrome 扩展项目
- 统一使用 `pnpm`

## 工作方式

- 用户负责：提要求、测试、反馈结果
- AI 负责：看代码、改代码、跑命令、处理细节、汇报结果
- 能直接做的事，AI 直接做，尽量不给用户加小操作
- 项目、版号、Git、项目文档这类内容，由 AI 主动维护
- 不确定的地方，AI 必须先问用户，禁止猜测

## 文档和记忆

- 本地规则、文档、日志这类内容，优先写在项目里，由 AI 主动维护
- `AGENTS.md` 这类本地文件，用来规定 AI 在这个项目里的工作方式
- `README.md` 这类项目说明文档，也由 AI 主动维护，方便接手的人或 AI 快速了解和开发项目
- MCP 记忆器是云端记忆，用来回忆用户偏好、历史约定和关键背景
- 一开始不清楚用户偏好时，AI 主动查看 MCP 记忆器
- 过程中遇到关键点，AI 主动判断是写到本地，还是记到 MCP

## 修改后要做的事

- 改了代码、功能、配置、打包相关内容后，默认执行 `pnpm build`
- 只改文档、规则、日志这类内容时，不用跑构建
- 除非用户明确说不用构建，否则代码改动后不能跳过
- 构建后要用中文汇报结果

## 构建汇报格式

- 构建结果：成功 / 失败
- 执行命令：`pnpm build`
- 如果失败：说明关键报错
- 如果失败：说明有没有继续尝试修复

## 沟通和安全

- 统一使用中文
- 说明简单直接，不长篇大论
- 未经用户明确同意，不执行危险的 Git 命令
- 不泄露本地密钥、账号、环境信息等敏感内容
