# 播放器问题修复清单

更新时间：2026-03-27

## 当前确认的问题

1. 左上角信息层未显示
- 期望：显示标题、文件大小、面包屑导航
- 现状：完全没有出现
- 处理策略：先不依赖现有静态节点挂载方式，改为运行时直接创建 overlay 节点并插入播放器根层，优先保证可见

2. 点击视频区域不能播放/暂停
- 期望：点击视频空白区域可切换播放/暂停
- 现状：点击无效，只能通过底部菜单控制
- 处理策略：单独接管视频点击行为，避免和播放器默认行为冲突

3. 出现 `undefined action!`
- 期望：进入播放和点击视频时不出现异常提示
- 现状：会弹出一行 `undefined action!`
- 处理策略：排查并移除触发该提示的默认 action / click 链路

4. 进入播放时存在“先闪一下再网页全屏”的感觉
- 期望：点击视频后直接进入网页全屏播放
- 现状：会短暂看到过渡状态
- 处理策略：放在前三项之后再处理，避免混淆主问题

- [x] 1. 修复左上信息层显示（标题、文件大小、面包屑）
- [x] 2. 修复点击视频区域切换播放/暂停
- [ ] 3. 消除 "undefined action!" 提示
- [ ] 4. 默认进入网页全屏且界面简化

## 工作方式

- 一次只修 1 项
- 每修完 1 项就构建并交给用户测试
- 用户确认后再进入下一项
